# Changelog

## [0.1.6] - 2026-06-10

- 공개 요약 API가 사이트 접속 시 OpenAI 요약을 생성하지 않고 저장된 요약만 반환하도록 변경
- Collector가 전날 요약과 최근 완료 주간 요약을 저장값이 없을 때만 생성하도록 수정
- 주간 요약 기준을 한국 시간 월요일 00시 이후 직전 월요일부터 일요일까지의 완료 주간으로 정리


## [0.1.5] - 2026-06-10

- 과거 RSS 기사도 현재 활성 언론사 기준으로 뉴스 검색에 노출되도록 복원
- RSS URL 정확히 일치 기준에 언론사 매핑 기준을 추가해 과거 기사 표시 범위 확대


## [0.1.4] - 2026-06-10

- 푸터에서 불필요한 로컬 테스트 문구 제거


## [0.1.3] - 2026-06-10

- 서비스 고지에서 개인용 모니터링 도구 표현을 공개 뉴스 브리핑 서비스 기준으로 수정
- 푸터 서비스 설명 문구를 RSS 기반 뉴스 브리핑 서비스로 정리

## [0.1.2] - 2026-06-10

- 웹 서버와 독립 실행되는 Collector 작업 스케줄러 배포 추가
- Collector 운영 배포 스크립트와 자동 재시작 실행 스크립트 추가

## [0.1.1] - 2026-06-10

- 사이트 푸터에 현재 배포 버전 표시

## [0.1.0] - 2026-06-10

- Pulse Brief의 첫 공식 버전 기준을 추가했습니다.
- RSS 수집, MongoDB 저장, 관리자 콘솔, OpenAI 요약, Cloudflare Tunnel 배포 구성을 포함합니다.
- 웹 서버와 수집기 분리 구조를 포함합니다.
